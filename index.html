code+wood toolkit

A list of some of my tools; Documented here mostly for my own use but feel free to used them if you find them useful. Documentation is sparse.

autocorrect

autocorrect is a fun but useless utility that will automatically try to match any methods or properties to the object you provide. it will find the _closest_ match. First character must match to the function All characters must be in the correct order. if it finds a match, it will bind the method to the object so it does not have to search again next time. just test out the example and look at the source. Dont use this at home. It's just fur fun.

canvas

I'm always messing around with canvas size, auto scale, aspect ratio, etc. this is my most recent template to remind me how it can be done nicely

colortools

colortools contain some helper functions to convert between named colors, hex colors and arrayus

it also has a function to create color quants to reduce the number of colors in an array by finding the closest match

svgtools

svgtools contain some helper functions to read and process svg files. It can simplify the svg and optimize the order of the paths
